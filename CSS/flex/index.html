flex-grow:

flex-grow: 1,2,3,4,5
Especifica que cantidad del espacio restante dentro del contenedor flexible, deberia de ocupar el item en cuestion, segun su direccion principal, esta ultima determinada por flex-direction
Ejemplo: si ponemos flex-grow: 1, al primer elmento este ocupara todo el ancho que sobre, pero si tenemos otro elmento con flex-grow: 2, este ultimo ocupara el doble del primero, pero respetando el ancho total, asi que el primer elemento se reduciria a la mitad del segundo
